<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration for teams, allowing multiple users to edit documents simultaneously via WebSockets on `wss://api.example.com/ws`.
- Introduced dashboard analytics with charts showing usage metrics.

## Improvements

- Optimized API response times by `>30%` for endpoints like `/v1/projects`.
- Enhanced search functionality with fuzzy matching.

## Bug Fixes

- Fixed authentication token refresh issues in long sessions.
- Resolved file upload errors for large attachments (`>100MB`).

</Update>

## Upgrading to the Latest Version

<Steps>
  <Step title="Backup Your Data" icon="database">
    Export your projects from the dashboard.

````bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.example.com/v1/export
````

  </Step>
  <Step title="Update Client" icon="download">
    Install the latest SDK.

    <CodeGroup tabs="npm,yarn">
    ````bash
npm install elyan-vadsaria@latest
````
    ````bash
yarn add elyan-vadsaria@latest
````
    </CodeGroup>
  </Step>
  <Step title="Test Changes" icon="check-circle">
    Run your integration tests against staging.
  </Step>
</Steps>

<Callout kind="tip">
  Subscribe to updates via email at `https://dashboard.example.com/notifications` to get alerts on new releases.
</Callout>
